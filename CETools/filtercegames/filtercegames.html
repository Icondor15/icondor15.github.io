Lists

Lists are a sub-tool of the CETools Game List which allow you to select games to put them in 'Lists' which can be easily shared via link!
(Hover over me to see all list info!)
Activate List Controls
(When list controls are active, checkboxes will appear next to each game's name. Selecting/deselecting these boxes will add/remove games from the list)
Creating a List: Click the checkboxes next to the games' names down below. The compressed ID and the games will update here for each game that gets added/removed.
Loading a List: Insert an ID in the 'Compressed ID' textbox below and click 'Load List' to load its contents! (Should happen automatically from a link.)
Sharing a List: Copy the link found at the bottom of this 'Lists' section, or copy the ID directly from the 'Compressed ID' section.
